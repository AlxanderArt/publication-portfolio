## Running the Project Locally

**Clone the repository**

```bash
git clone https://github.com/AlxanderArt/publication-portfolio.git
```

**Install dependencies**

```bash
npm install
```

**Run the development server**

```bash
npm run dev
```

**Open in your browser**

```
http://localhost:3000
```
